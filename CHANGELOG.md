# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), this project
follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html), and entries are generated
automatically on each release from [Conventional Commits](https://www.conventionalcommits.org/)
(`feat:` → minor, `fix:` → patch, `feat!:`/`BREAKING CHANGE` → major).
